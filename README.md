# VRC-Photos-Pro
Capturing VRChat memories—more easily, more vividly.

VRC-Photos-Pro is a lightweight, high-speed desktop utility built with Tauri (Rust) and React designed to dramatically streamline how you manage and share your VRChat photography.

## Key Features
### Real-time Photo Sync: 
Automatically monitors your VRChat folder and imports new captures instantly.

### Metadata Analysis: 
Automatically extracts embedded metadata, including World Names and timestamps.

### Easy SNS Sharing: 
Upload managed photos directly to social media from the app.

### Note: 
This uses the X API (Free Tier), so please consider it a secondary feature as it may occasionally hit rate limits.

### Auto-Updates: 
Stay up to date with the latest features and security fixes via a custom-built Tauri updater.

### World History Logging: 
Never lose track of where your favorite memories were made.

## Installation
Navigate to the Releases page.

Download and run VRC-Photos-Pro_0.2.0_x64_en-US.msi.

Once installed, launch the app via the desktop shortcut.

## Tech Stack
Frontend: React / TypeScript / Vite

Backend: Rust (Tauri)

Sidecar: get_meta (Custom binary for metadata extraction)
